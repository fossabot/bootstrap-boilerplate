# bootstrap-boilerplate
Project to start development with Gulp, Bootstrap and its dependencies.sta

## Installation
```
$ git clone ssh@github.com:oeduardoal/bootstrap-boilerplate.git project`
```

## Starting
### Install packages
```
cd project && npm i
```

## Gulp
```
$ gulp
```
